# 🍽️ Vishwas Sweets

Vishwas Sweets is a full-stack restaurant management and food ordering platform where customers can **browse menus**, **place orders**, **reserve tables**, and **manage their food cart seamlessly**.

---

## ✨ Features

### 👤 Authentication

* User registration and login
* Secure JWT authentication
* Protected user routes
* Admin authorization for restaurant management

### 🍛 Menu Management

* Browse food categories
* View detailed menu items
* Food images with Cloudinary integration
* Search and filter menu items

### 🛒 Cart System

* Add items to cart
* Update item quantities
* Remove items from cart
* View total bill before checkout

### 📦 Order Management

* Place food orders online
* Track order status
* View order history
* Admin order management dashboard

### 🪑 Table Booking

* Reserve tables online
* Select booking date and time
* Manage booking requests
* Admin booking approval system

### 📂 Category Management

* Create food categories
* Update category information
* Delete categories
* Organize menu items efficiently

### 🛠 Admin Dashboard

* Manage categories
* Manage menu items
* Manage customer orders
* Manage table bookings
* Restaurant analytics and monitoring
